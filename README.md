# NPC 홈페이지
로컬 환경에서 실행을 원하면, [Docker를 설치](https://docs.docker.com/get-docker/)하고 다음 명령어를 입력해주세요.

```sh
$ git clone https://github.com/skku-npc/website.git
$ cd website
$ docker-compose up -d
```

이후 http://localhost:3000/ 에 접속하면 됩니다.
